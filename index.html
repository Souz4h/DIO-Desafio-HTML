Surgimento da Internet

< Voltar

A internet surgiu na década de 1960, durante a Guerra Fria, com o projeto ARPANET, criado pelo Departamento de Defesa dos EUA para uma rede resistente a ataques. A ARPANET utilizava comutação de pacotes, dividindo dados em partes para transmissão eficiente.

Em 1969, a primeira conexão ocorreu entre computadores na Universidade da Califórnia em Los Angeles e o Instituto de Pesquisa de Stanford. Nos anos 1970, o protocolo TCP/IP, de Vint Cerf e Robert Kahn, tornou padrão a comunicação entre redes.

Ambas as formas de comunicação são fundamentais para o funcionamento adequado de aplicativos web e são executadas através de diferentes tecnologias, como HTML, JavaScript, APIs e bancos de dados, entre outros.

HTML

< Voltar

HTML (HyperText Markup Language) é a linguagem usada para construir a estrutura de páginas na internet e serve como o alicerce de qualquer site ao definir elementos como textos, fotos, links, botões e muito mais.

Com a linguagem HTML é possível estruturar e apresentar informações com clareza através da utilização elementos como:

  • <h1> a <h6>: Títulos de diferentes níveis
  • <p>: Parágrafos
  • <a>: Links
  • <img>: Imagens
  • <div> e <section>: Estruturação de conteúdo
